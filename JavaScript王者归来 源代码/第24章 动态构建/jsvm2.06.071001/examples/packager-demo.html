Example: JSVM Class Packager!

　

  1. 声明：该工具需要 FileSystemObject 组件的支持！所以使用改工具的环境必须是 IE. 并且如果访问的不是本地文件而是通过HTTP访问的页面，必须对该url进行信任站点授权操作。

  2. 在 JSVM 的环境配置中，设置 modules 参数，加入：runtime,tools, （tools依赖于runtime）。例如： <script src="jsvm2/jsre.js" modules="runtime,tools"></script>

  3. 使用热键 ctrl+F9 调出 packager 的程序界面！

  4. 程序提供三种打包方式

    • 列出所有ClassHome下的class.js文件，将选中的文件打包输出到一个指定的路径。
    • 列出指定的某个现成的包文件中的所有class.js名称，将选中的文件打包输出到一个指定的路径。
    • 列出JSVM所有已经加载过的class.js文件，将选中的文件打包输出到一个指定的路径。

  5. 将输出的文件拷贝到<jsvm目录>/lib 目录中，在jsvm的环境变量classpath中配置进去。例如： <script src="jsvm2/jsre.js" classpath="lib文件1;lib文件2"></script> （多个lib文件之间;分割）
